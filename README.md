# FoodExplorer 🍴

A recipe search app built with React and TypeScript, powered by TheMealDB API. Browse recipes from around the world, filter by category, and view detailed ingredients and instructions.

## 🚀 Live Demo

_Live demo:_ [food-explorer-mpdev.vercel.app](https://food-explorer-mpdev.vercel.app)

## 🛠️ Tech Stack

- **React** + **TypeScript**
- **Vite**
- **Tailwind CSS**
- **TheMealDB API**

## ✨ Features

- Browse recipes by category
- Search recipes by name
- View detailed recipe information — ingredients and instructions
- Split-view layout — meal list and detail panel side by side
- Responsive design — works on mobile and desktop
- Empty state handling

## 📦 Installation

```bash
git clone https://github.com/michalpawlowski-mp/food-explorer.git
cd food-explorer
npm install
npm run dev
```

## 📡 API

This project uses [TheMealDB](https://www.themealdb.com/api.php) — a free, open meal database. No API key required.

Base URL: `https://www.themealdb.com/api/json/v1/1`
